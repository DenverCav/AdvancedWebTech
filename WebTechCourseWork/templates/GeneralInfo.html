General Information

This site is being made for a University Project. It is intended for use by the Web Tetris community as a way to have a more visually appealing viewing and user experience as compared to the standard Google Sheet that is currently in use by the community.
This site is using the data directly from the standard Google Sheet and simply reformatted allowing for new features such as toggles and filters to view the data in different ways.
All the data is being used with permission from the Leaderboard team in the Tetris Tryhards Discord server
